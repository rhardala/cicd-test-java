0/ Prérequis


----------- Git / GitHub: -----------

sudo yum install git
git --version  (pour vérifier l'installation)

Eventuellement créer un compte sur github.com si vous n'en avez pas un

Générer un Personal Access token:

User ==> settings ===> glisser tout en bas à gauche: Developer Settings
==> Personal Access Token ===> donner un nom au token (Ex: Centos AWS for CICD) ===>
Générer le token, le copier et le GARDER JALOUSEMENT!!!


----------- Java 11 : -----------


sudo yum install java-11-openjdk-devel

java version (pour vérifier l'installation)


----------- Maven: -----------

sudo yum install maven
mvn --version (pour vérifier l'installation)

Forcer maven à utiliser java 11

sudo alternatives --config java (puis taper 1 + entrer)

sudo alternatives --config javac (puis taper 1 + entrer) : Ce n'est pas une repétition, il y'a un 'c' après java
